HOW TO GET TEST AUDIO FILES
===========================

Option 1: Use your own meeting recording
-----------------------------------------
Place any audio file (MP3, WAV, M4A, OGG, FLAC) in this folder.

Option 2: Record a quick test
-------------------------------
1. Open Windows Voice Recorder
2. Record yourself saying something like:
   "This is a test meeting. We discussed the project timeline.
    John will complete the report by Friday."
3. Save as test_meeting.m4a or test_meeting.mp3 in this folder

Option 3: Download a sample
----------------------------
Visit: https://www.kozco.com/tech/organfinale.wav
Download any public domain audio and save it here.

Option 4: Use online TTS
------------------------
1. Visit: https://ttsmaker.com/
2. Paste meeting text
3. Generate and download MP3
4. Save in this folder

Then run: python main.py summarize audio_input/YOUR_FILE.mp3
