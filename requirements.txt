# ============================================================================
# NOTES FOR INSTALLATION
# ============================================================================
#
# 1. Install base requirements:
#    pip install -r requirements.txt
#
# 2. For Whisper to work, you also need ffmpeg:
#    - Windows: choco install ffmpeg (or download from ffmpeg.org)
#    - macOS: brew install ffmpeg
#    - Linux: apt-get install ffmpeg
#
# 3. For LLM support with Ollama:
#    - Download Ollama from https://ollama.ai
#    - Run: ollama pull mistral (or your desired model)
#    - Start with: ollama serve
#
# 4. For HuggingFace Transformers backend:
#    - Models will auto-download on first use
#    - Requires ~4-7GB disk space for Mistral-7B
#    - Requires 4-8GB VRAM for inference
#
# ============================================================================
